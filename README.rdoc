This is a {Chef}[http://wiki.opscode.com/display/chef/Home] Cookbook repository following the {Headless Branches}[http://wiki.opscode.com/display/chef/Headless+Branches+for+Cookbook+Repositories] pattern.

Most of the cookbooks are based on or borrowed from other repositories, like {opscode/cookbooks}[http://github.com/opscode/cookbooks/], {37signals/37s_cookbooks}[http://github.com/37signals/37s_cookbooks/], and {nbudin/chef-repo}[http://github.com/nbudin/chef-repo/tree/master/cookbooks/].
