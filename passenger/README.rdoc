= DESCRIPTION:

Installs and configures Passenger with nginx

= REQUIREMENTS:

== Platform:

Tested on Ubuntu 9.10. May work on other platforms, esp Ubuntu/Debian.

== Cookbooks:

* ruby
* nginx

= ATTRIBUTES: 

* passenger[:version] - Specify the version of passenger to install.
* passenger[:root_path] - The location of the passenger gem.

= USAGE:

For example, to run a Rails application on passenger:

  include_recipe "passenger::nginx"
